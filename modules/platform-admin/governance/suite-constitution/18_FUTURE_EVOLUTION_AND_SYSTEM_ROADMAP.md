# 0) Roadmap Principles (Non-Negotiable)

## 0.1 Core Stability First

Core contracts and tenant safety remain stable while new features are built externally via Suites and Extensions.

## 0.2 Backward Compatibility

Future changes must preserve:

- API contracts
- tenant data integrity
- extension compatibility

## 0.3 Incremental Evolution

The platform grows in phases rather than via disruptive redesigns.
